# Бизнес-план: ML Loan Service

## 1. Описание продукта

**ML Loan Service** — облачный API-сервис для автоматической оценки кредитоспособности заёмщиков на основе машинного обучения. Банки, МФО и финтех-компании подключаются к API и получают мгновенный скоринг без необходимости разрабатывать собственные ML-системы.

## 2. Уникальное торговое предложение (УТП)

| Что предлагаем | Зачем это клиенту |
|---|---|
| Готовый ML-скоринг через REST API | Экономия 6-12 месяцев разработки |
| Оплата за результат (кредиты) | Нет фиксированной подписки, платишь только за запросы |
| Точность модели 93% (GradientBoosting) | Снижение процента одобрения "плохих" заёмщиков |
| Промокоды и бонусные кредиты | Стимулирование активных пользователей |
| Асинхронная обработка (Celery + Redis) | Не блокирует клиентский сервис при высокой нагрузке |
| Мониторинг в реальном времени | Grafana дашборд с метриками предсказаний и расхода кредитов |

## 3. Целевая аудитория

- **Микрофинансовые организации (МФО)** — нужен быстрый скоринг без штатного ML-инженера
- **Небольшие банки и кредитные кооперативы** — хотят улучшить автоматизацию одобрения
- **Финтех-стартапы** — MVP без инвестиций в ML-инфраструктуру

## 4. Финансовая модель

### Структура монетизации

Система кредитов (внутренняя валюта):
- 1 кредит = 1 запрос предсказания
- Кредиты покупаются пакетами

### Тарифные пакеты

| Пакет | Кредитов | Цена | Цена за запрос |
|---|---|---|---|
| Старт | 1 000 | 500 руб. | 0.50 руб. |
| Бизнес | 10 000 | 3 000 руб. | 0.30 руб. |
| Корпоративный | 100 000 | 20 000 руб. | 0.20 руб. |

### Промокоды для привлечения

- `WELCOME100` — 100 бесплатных кредитов при регистрации
- Реферальные промокоды для партнёров

### Прогноз выручки (первый год)

| Месяц | Клиентов | Запросов/мес | Выручка |
|---|---|---|---|
| 1-3 | 5 | 2 000 | 6 000 руб. |
| 4-6 | 20 | 15 000 | 45 000 руб. |
| 7-12 | 50 | 60 000 | 150 000 руб. |

### Структура затрат

| Статья | Стоимость/мес |
|---|---|
| Сервер (VPS 4 CPU, 8GB RAM) | 3 000 руб. |
| PostgreSQL + Redis (cloud) | 1 500 руб. |
| Monitoring (Grafana Cloud) | 0 руб. (free tier) |
| Итого | ~4 500 руб. |

**Точка безубыточности:** ~9 000 запросов в месяц (при средней цене 0.50 руб./запрос = ~4 500 руб. выручки)

## 5. Конкурентный анализ

| Критерий | ML Loan Service | AWS SageMaker | Самостоятельная разработка |
|---|---|---|---|
| Стоимость входа | Низкая | Высокая | Очень высокая |
| Время внедрения | 1 день | 2-4 недели | 6-12 месяцев |
| Кастомизация | Администратор загружает модели | Полная | Полная |
| Поддержка | API документация | Корпоративная | Своя команда |

## 6. План развития

**Этап 1 (сейчас):** MVP — скоринг кредитов, базовый биллинг, промокоды

**Этап 2:** Реферальная программа, webhook-уведомления о результатах, SDK для Python/JS

**Этап 3:** Маркетплейс моделей (клиенты продают свои обученные модели другим клиентам)

## 7. Ключевые риски

| Риск | Митигация |
|---|---|
| Точность модели недостаточна | Возможность загрузки собственной модели клиентом |
| Утечка данных заёмщиков | Данные не хранятся, только input/output в JSON |
| Конкуренты | Фокус на простоте интеграции и цене |
